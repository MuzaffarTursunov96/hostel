# 🏨 Hotel Management System

Full-stack hotel management system.

## Features
- Room & bed management
- Active & future bookings
- Desktop app (PySide6)
- Web mini app (Flask)
- Telegram bot
- REST API (FastAPI)

## Tech Stack
- Python, FastAPI, Flask
- SQLite / PostgreSQL
- Docker & Nginx
- PySide6
- Telegram Bot API

## Run Backend
```bash
cd backend
uvicorn main:app --reload
